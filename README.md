# Analisis-Descriptivo-de-los-Accesos-Fijos-a-Internet-por-Proveedor-y-Segmento-en-Colombia-2022-2025
El presente análisis se desarrolló a partir de un conjunto de datos del sector de Tecnologías de la Información y las Comunicaciones (TIC), enfocado en el estudio del acceso a internet fijo, las velocidades de conexión y las tecnologías asociadas.
